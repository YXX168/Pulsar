# 更新日志

本项目采用 `主版本.次版本.修订版本` 的版本格式。完整安装包与自动生成的提交记录可在 [Releases](https://github.com/YXX168/Pulsar/releases) 查看。

## 1.11.1

- 将脉冲矩阵星期标签改为七等分固定布局，修复周日裁切与切换移位。
- 为星期和周次标签加入平滑渐变、缩放与淡入动画。
- 将星环模式周次切换移动到日星球下方。

## 1.11.0

- 重做脉冲星总核心符号和倒计时能量轨。
- 修复本地 JSON 文件备份与恢复，并重做状态提示。
- 新增周次浏览、仅本周跳过和回到本周。
- 新增重量、次数、RPE、备注等详细单组记录。
- 新增训练报告以及任意单组编辑、删除功能。
- 原生适配 Android 高刷新率显示模式。

## 更早版本

早期版本逐步完成了星环裂变、项目光球、脉冲矩阵、训练记录、计划设置、Android 返回链路与云端 APK 发布。详情请查看 [Release 历史](https://github.com/YXX168/Pulsar/releases)。
