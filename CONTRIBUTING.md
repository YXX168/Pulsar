# 贡献指南

感谢你愿意帮助维护 Pulsar。

## 提交问题

- Bug 请写明应用版本、设备型号、Android 版本和可复现步骤。
- 动画或布局问题建议附带录屏或截图。
- 功能建议请说明使用场景，而不只是描述按钮或页面。
- 请勿在公开 Issue 中上传备份文件、签名文件或任何敏感数据。

## 提交代码

1. Fork 仓库并从 `main` 创建功能分支。
2. 保持界面简洁，避免引入与星海视觉不一致的通用组件。
3. 数据模型变更必须兼容旧版本本地数据和 JSON 备份。
4. 提交前运行：

```bash
dart format lib test
flutter analyze
flutter test
```

5. Pull Request 中说明改动目的、验证方式和可能影响，视觉变更请附前后对比。

## 设计原则

- 动画应连贯、可逆，并在 120Hz 设备上保持稳定。
- 光球、文字和控制项不得互相遮挡。
- 普通点击保持快速，详细记录使用长按或次级入口。
- 本地优先，不添加广告、追踪或不必要的网络权限。
