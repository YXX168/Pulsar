# PULSAR

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

一款纯 Flutter 打造的训练星系与健身记录应用。视觉灵感来自脉冲星、液态流光和轨道系统，但拥有独立的交互与视觉设计。

## 核心体验

- 「星环模式」与「脉冲矩阵」两种显示方式，共享同一份计划与进度
- 总能量核心裂变为七颗日星球，再进入当天训练星群
- `CustomPainter` 绘制液态能量、体积雾、玻璃高光与动态星环
- 每个训练或成长事项是一颗子星球，每点击一次完成一组
- 动作完成后再点击两次即可清零，无额外刷新按钮
- 周六支持阅读、学习、AI 创造、恢复与自定义训练事项
- 可浏览历史/未来周次，并支持“仅本周跳过”且不改动长期模板
- 长按训练项目可精确记录实际次数、重量、RPE 与单组备注
- 训练报告支持按动作查看、编辑和删除任意一组，完成度同步回退
- 日期化进度、永久事项 ID、训练容量、个人纪录和一键撤销
- 可暂停、追加 15 秒的组间休息计时，以及计划复制与排序
- SQLite 本地数据库、旧版本自动迁移以及系统文件式 JSON 完整备份/恢复
- 静谧、流畅、极致三档动态效果
- 原生 Android 震动反馈、Flutter 页面转场动画及高刷新率模式

## Android APK

每次推送都会由 GitHub Actions 执行分析、测试和 Android 构建。版本标签会自动发布通用 APK，以及体积更小的 arm64-v8a、armeabi-v7a 和 x86_64 安装包。
